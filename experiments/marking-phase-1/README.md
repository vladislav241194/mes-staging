# MES-line · Маркировка · интерактивная фаза 1

Изолированный React + TypeScript прототип рабочего места маркировки. Он не подключён к runtime MES, API, PostgreSQL, авторизации, Shift Execution или физическому принтеру.

## Локальный запуск

```bash
cd /Users/vladislav/Documents/Codex/2026-05-30/mes-marking-phase-1/experiments/marking-phase-1
npm install
npm run dev -- --port 4179
```

Открыть: `http://127.0.0.1:4179/`.

Проверки:

```bash
npm run check
```

## Границы прототипа

- Всё состояние хранится в `localStorage` под ключом `mes-line:marking-phase-1:test-state:v1`.
- Кнопка «Сбросить тест» возвращает исходные тестовые задания.
- Печать проходит только через `testPrintAdapter`; физическое оборудование и сетевые сервисы не вызываются.
- Сканер работает в режиме просмотра. Камера — интерактивный макет без распознавания.
- Производственные сущности представлены реалистичными тестовыми проекциями: задание, изделие, СЗН, сотрудник, участок, комплекты и история.
- Pilot и реальные производственные данные не используются.

Подробный визуальный и функциональный отчёт: [reports/visual-review.md](reports/visual-review.md).
