# UI Typography Matrix

Актуальная краткая матрица модулей после удаления устаревшего UX-модуля. Подробные значения шрифтов и состояний проверяются через `UI-состояния`, `scripts/design-qa-snapshots.mjs` и встроенную аннотацию браузера Codex.

| Группа | Модули | Базовый UI-паттерн |
|---|---|---|
| Технологии | Маршрутная карта, Спецификации, Номенклатура | `module-data-page`, внутренний сайдбар, плотные таблицы, tree-cell для структуры |
| Планирование нагрузки | Планирование, Заказ-наряды | Gantt shell, planning table, структурный документ заказ-наряда |
| Оперативное управление | Диспетчерская, Мастерская | оперативные панели, контекстная рабочая область, сменные листы |
| Система | Сотрудники, Справочники | оргструктура, мастер-данные, таблицы с фильтрами |
| UX-макеты | UI-состояния, План-таблица, Снабжение, Цех производства | тестовая группа, визуальные прототипы без влияния на производственный план |

## Контракт

- Базовый текст рабочих модулей: 11-12px, без необоснованной жирности.
- Сайдбары используют единый `module-data-sidebar`.
- Таблицы используют внутренний scroll только для реальной плотной таблицы или временной шкалы.
- Старые модули и удаленные сущности не должны добавляться в матрицу.
