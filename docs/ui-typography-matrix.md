## Наблюдения

| Зона | Что видно по матрице | Риск |
|---|---|---|
| Общий typography contract | Большинство production-модулей сведены к шкале 9 / 9.5 / 10.5 / 11 / 13 / 15 / 17 px. | Хорошая база, но поздние локальные правила отдельных модулей могут перебивать contract. |
| Гант | Имеет отдельные правила для topbar, operation slots, dependency labels и transfer markers. | Самый высокий риск визуального рассинхрона: много специальных исключений и absolute/SVG слоев. |
| Диспетчерская | Использует локальные переменные `--dispatch-font-xs/sm/md/lg`, частично совпадает с общей шкалой. | Может начать жить отдельной типографикой, если не привязать переменные к `--mes-ui-type-*`. |
| Справочники | Внутренний сайдбар выглядит как эталонный паттерн для навигации. | Его активные/пассивные состояния стоит формализовать как общий компонент sidebar-list. |
| Карта цеха | Модуль есть в render-ветке, но не входит в `getModuleGroups`. | При возврате в меню нужен отдельный визуальный аудит, сейчас пользовательский путь до него отсутствует. |
| РКД | UX-only/test: стили маркируют тестовую группу и отключенный сервис. | Нельзя смешивать его визуальный язык с production-модулями без явного test-маркера. |

| Компонент | Единый стандарт |
|---|---|
| Главный сайдбар | group title 8.5-9.5px / nav 10.5px / active state один на все группы. |
| Внутренний сайдбар | title 15px / item strong 10.5px 600 / meta 9.5px 500 / badge 9.5px 600. |
| Верхнее меню | app title 17px 700 / controls 11px 600 / container heights фиксировать токеном. |
| Карточки | card title 13px 700 / body 11px 400 / caption 9.5px 600. |
| Таблицы | th 11px 600 / td 11px 400 / primary cell 11px 600. |
| Бейджи и статусы | 9.5-11px, без локальных прыжков веса выше 700. |
