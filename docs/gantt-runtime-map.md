# Gantt Runtime Map

The Gantt module is a special runtime. It is not a normal table or panel layout because it combines absolute positioning, timeline scale calculation, SVG dependencies, drag/resize affordances and modal/drawer overlays.

Primary files:

- `src/app.js`
- `src/gantt_ui_contracts.js`
- `styles/mes-ui-core.css`
- `styles/layers/10-shell-directory-gantt-base.css`
- `styles/layers/40-gantt-planning-routes.css`
- `scripts/gantt-ui-regression-smoke.mjs`
- `scripts/gantt-inline-style-audit.mjs`
